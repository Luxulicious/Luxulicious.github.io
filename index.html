C# Unity3D Game Dev Game Tool February 2020 Ongoing

SOA (Scriptable Object Architecture)

Description
A Unity package for creating a data-driven based architecture within Unity.

Check it out here on Github!

Features
  • Variables as Scriptable Objects
  • References that can be used within scripts to reference either a variable, locally or a constant
  • Game Events as Scriptable Objects which function similar to Unity Events
  • Game Event Listeners for listening to game events from within scripts Tool for generating variables, references, game events and listeners
  • OdinInspector/-Serializer version coming soon!

Highlights
  • A generic framework which is easy to implement
  • Custom inspector/UI work
  • Code generation via templates
